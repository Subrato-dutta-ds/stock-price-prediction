# 📈 Stock Price Prediction System

A Machine Learning based web application that predicts stock prices using historical market data and deep learning techniques. The project provides stock trend visualization, future price prediction, and an interactive dashboard built with Streamlit.

---

## 🚀 Features

- 📊 Real-time stock market data visualization
- 🤖 Machine Learning based stock price prediction
- 📈 Interactive charts and trend analysis
- 🧠 Deep Learning model integration using TensorFlow/Keras
- 🌐 User-friendly Streamlit web interface
- 📅 Historical stock data analysis

---

## 🛠️ Tech Stack

- Python
- Pandas
- NumPy
- TensorFlow / Keras
- Scikit-learn
- Streamlit
- Matplotlib
- Yahoo Finance API

## ⚙️ Installation & Setup

### 1️⃣ Clone Repository

```bash
git clone https://github.com/Shadowgit21/stock-price-prediction.git
```

### 2️⃣ Navigate to Project Folder

```bash
cd stock-price-prediction
```

### 3️⃣ Install Dependencies

```bash
pip install -r requirements.txt
```

### 4️⃣ Run Application

```bash
streamlit run app.py
```
